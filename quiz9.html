🔐 Two-Factor Authentication

You use the same password "MyFavoriteColor123" for your email, social media, and games. Is this a good idea?
⚠️
Using the Same Password
"I use 'MyFavoriteColor123' for everything - my email, Instagram, TikTok, and my games. It's so much easier to remember just one password!"
💥
BIG DANGER:
• If one gets hacked, they all get hacked • Bad people get into everything • All your stuff gets stolen
💡
Remember: Different Passwords for Each Thing!

Using different passwords keeps you safe if one account gets hacked.

✅ Yes, this is smart - one password is easier to remember
🚫 No, this is dangerous - I should use different passwords for each thing
🤔 It's okay as long as the password is strong
